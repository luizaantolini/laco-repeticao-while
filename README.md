# laco-repeticao-while

Este repositório contém cinco programas JavaScript que utilizam estruturas de repetição para resolver os exercícios propostos.

## Exercícios implementados 👻🪼

1. `ex01` - solicita um número positivo e continua pedindo quando o número for negativo.
2. `ex02` - recebe notas de 0 a 10 até receber um número negativo e exibe a média.
3. `ex03` - lê valores de compras e exibe o total quando o usuário digita 0.
4. `ex04` - pede primeiro nome e sobrenome separadamente, repetindo o pedido quando algum campo ficar em branco.
5. `ex05` - simula um saque em conta com saldo inicial e repete o pedido até o saque ser possível; encerra quando o saldo chega a 0 ou negativo.
